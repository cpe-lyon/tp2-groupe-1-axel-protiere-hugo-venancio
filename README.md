## Exercice 1. Variables d'environnements

1. Dans quels dossiers bash trouve-t-il les commandes tapées par l'utilisateur?   
Dans les dossiers suivants :  
/usr/local/sbin    
/usr/local/bin  
/usr/sbin  
/usr/bin  
/sbin  
/bin  
/usr/games  
/usr/local/games  
/snap/bin  
Ils sont définis par la variable d'environnement PATH
